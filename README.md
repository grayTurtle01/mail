## Setup 🖥️
`git clone https://github.com/grayTurtle01/mail.git`

`cd mail`

`pip install -r requiremets.txt`

`python manage.py makemigrations`

`python manage.py migrate`

`python manage.py runserver`

`localhost:8000`
